Para instalar e executar o sistema favor seguir os seguintes passos

### Usando a aplicacao
Backend para codigo rest client onde deve rodar :
###### Para Instalar Dependencias
`mvn install`
###### Para Rodar
`mvn spring-boot:run` 
###### Para Executar Testes Unitarios
`mvn test`
